Búsqueda de Usuarios ⏷


¿Cómo realizo una búsqueda?

Ingresa un nombre de usuario o dirección IP en la barra superior.

Luego presiona Enter o haz clic en el icono de la lupa para buscar.

¿Qué hago si no me encuentra el nick que busco?

Prueba con otros nombres similares o cuentas relacionadas.

Es posible que el usuario no esté en la base de datos todavía.
